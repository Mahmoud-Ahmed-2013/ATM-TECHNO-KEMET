# ATM-TECHNO-KEMET

الفريق
- محمود (قائد الفريق)  
- أحمد  
- آدم  

وصف المشروع
ATM-TECHNO-KEMET هو نظام ماكينة صراف آلي (ATM) مطور بواجهة حديثة، يهدف إلى تحسين تجربة المستخدم من خلال دمج تقنيات الذكاء الاصطناعي والتعرف على الوجه.  
المشروع يحاكي عمل ماكينات الـATM التقليدية مع إضافة خصائص مبتكرة لزيادة الأمان وسهولة الاستخدام.  

التقنيات المستخدمة
- Teachable Machine → لتدريب نموذج التعرف على الوجه.  
- App Lab (Code.org) → لتصميم الشاشات والتجربة التفاعلية الأولية.  
- Visual Studio Code → لتطوير الكود النهائي وربط الخصائص.  

المميزات الأساسية
- تسجيل الدخول باستخدام التعرف على الوجه لزيادة الأمان.  
- واجهة رئيسية سهلة الاستخدام مع شاشات فرعية للخدمات.  
- خدمات أساسية:  
  - سحب النقود  
  - إيداع الأموال  
  - الاستعلام عن الرصيد  
- نظام مساعد صوتي/شات بوت لتوجيه المستخدم.  
- إشعارات للتأكيد على العمليات المنفذة.  

الخصوصية والأمان
- لا يمكن لأي طرف غير المستخدم والبنك معرفة الرصيد.  
- كلمة السر مبنية على ترميز ASCII للحروف (A=65, M=77) كاختصار لأعضاء الفريق (Adam, Ahmed, Mahmoud).  

خطوات التشغيل
1. تشغيل المشروع من خلال Visual Studio Code.  
2. اختيار الخدمة (إيداع / سحب / استعلام).  
3. استخدام الشات بوت للتوجيه.
